# システム要件

<Callout emoji="💻">以下は <strong>推奨動作環境</strong> です。動作を確実に保証するものではありません。</Callout>

## 推奨スペック

| デバイス | OS バージョン | CPU | メモリ | GPU | 備考 |
|---------|--------------|------|-------|------|------|
| Mac | macOS 12+ (Monterey) / Apple Silicon | **M1** 以上 | **8 GB+** | 内蔵 GPU で可 | Chrome ベースブラウザ |
| Windows | Windows 10 21H2+ / 11 | **第12世代 Core i5** 以上 | **8 GB+** | 内蔵 GPU で可 | Chrome ベースブラウザ |
| Android | Android 14+ (Tensor G1 以降) | ― | 4 GB+ | - | Chrome ベースブラウザ |
| iPhone | iOS 16+ (iPhone 12 以降) | ― | 4 GB+ | - | Chrome ベースブラウザ |

### GPU について

本シミュレーターは軽量設計のため、高性能 GPU を必要としません。CPU 内蔵 GPU で問題なく動作します。
